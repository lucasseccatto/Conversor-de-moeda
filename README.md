# Conversor de Moeda

## 📝 Apresentação:
Projeto para converter moeda, onde o usuário pode fazer a conversão para USD, Euro e Bitcoin tempo real.
<br>
API consumida: AwesomeAPI (https://docs.awesomeapi.com.br/api-de-moedas)
